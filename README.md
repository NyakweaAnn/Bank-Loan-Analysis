# 🏦 Loan Approval Predictive Analysis (EDA)
This project focuses on identifying the key drivers behind bank loan approvals using a dataset of 614 applicants. Through data cleaning, feature engineering, and exploratory data analysis (EDA), I uncovered critical insights into how credit history, geography, and household income influence lending decisions.
# Quick Insights
-Credit is King: Credit History is the #1 predictor. No history = almost certain rejection.
-Wealth Paradox: Rejected applicants earn $7,324 on average vs. $6,889 for approved—proving income doesn't fix bad credit.
-Top Market: Semiurban areas have the highest approval efficiency.
# Technical Summary
Cleaning: Used Statistical Mode to fix missing values in Gender, Marriage, and Dependents.

Engineering: Created Total_Income to measure household "Repayment Power."

Visualization: Built an Dashboard with interactive Slicers.
# Recommendation
Target high-earning applicants with "Credit Builder" products to capture the market currently being rejected despite high salaries.
